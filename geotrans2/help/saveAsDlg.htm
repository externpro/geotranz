GEOTRANS Save As Dialog Box

Output files should be saved as text files (.txt or .dat). 

The Windows version of the GEOTRANS Save As dialog box is a standard Windows Save As dialog box, which is identical in layout to the File Open dialog box. This dialog box contains the following elements:

  • a 'Look in:' pull down list that allows you to move to higher levels in the directory hierarchy,
  • a button that allows you to move up one level in the directory hierarchy,
  • a button that allows you to create a new folder,
  • a pair of buttons that controls the format of the displayed files and folders (List vs. Details),
  • a window showing the files and folders that are contained within the current folder, which you can use to select the file to be opened,
  • a 'File name:' field, in which you can enter the name of the file to be opened,
  • a 'Files of Type:' pull-down list that allows you to filter the displayed files by their type,
  • an Open button, which causes the currently selected file to be opened, and the GEOTRANS File Processing window to appear,
  • a Cancel button, which aborts file processing and returns you to the GEOTRANS main window.

The Java version of the GEOTRANS Save As dialog box is a standard Java Save As dialog box.  Its layout and functionality are essentially identical to the standard Windows Save As dialog box.

The Motif version of the GEOTRANS Save As dialog box is a standard Motif Save As dialog box, which is identical in layout to the File Open dialog box. This dialog box contains the following elements:

  • a Filter field, in which you can enter a path name and a partial file name, including wildcard characters, to filter the list of files displayed,
  • a scrolling list of the currently accessible Directories, which you can use to move around in the directory hierarchy,
  • a scrolling list of the Files located within the current directory, which you can use to select a particular file to be opened,
  • a Selection field, in which you can enter the path and name of the file to be opened, and
  • a row of four buttons across the bottom:

-        OK – which causes the currently selected file to be opened, and the GEOTRANS File Processing window to appear,

-        Filter – which causes the displayed list of files to be filtered according to the contents of the Filter field,

-        Cancel – which aborts file processing and returns you to the GEOTRANS main window, and

-        Help – which activates the on-line help subsystem.
