Delete Ellipsoid Dialog Box

The Delete Ellipsoid dialog box contains the following subelements:

  • an Ellipsoid pull-down list, with which you specify the user-defined ellipsoid that is to be deleted.
  • a Cancel button – which aborts ellipsoid deletion and returns you to the GEOTRANS main window.
  • an OK button – which causes the specified user-defined ellipsoid to be deleted.
