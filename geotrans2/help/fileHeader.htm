Input File Header Generation

Although the GEOTRANS coordinate file format, described in the next section, is really quite simple, creating the header for an input coordinate file manually can be an error prone process.  Fortunately, there is a simple way to generate input coordinate file headers automatically, using the GEOTRANS file processing GUI, as follows:

1.   Select any existing GEOTRANS coordinate file, as described in Section 4.2.1.  Example coordinate files can be found in the /examples subdirectory.

2.   In the output portion of the GEOTRANS File Processing window, specify the datum, the coordinate reference system type, and any associated parameters, as described in Section 4.2.2, for the input coordinate file that you desire.

3.   Specify a file name and directory for the resulting output file, as described in Section 4.2.3.

4.   In the GEOTRANS File Processing dialog box, click on the Cancel button to abort file processing.

The resulting output file will contain the specified header information.  This header can then be attached to an existing text file of coordinates to create a GEOTRANS input coordinate file.

The GEOTRANS Java GUI includes a command in the File pull-down menu, Create File Header, which allows this process to be performed without using an existing GEOTRANS coordinate file as input.
